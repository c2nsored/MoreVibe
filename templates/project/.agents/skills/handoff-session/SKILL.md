# Handoff Session

## Goal

Leave the next session with enough context to continue without re-discovery.

## Include

- current status
- latest changes
- remaining risks
- recommended next action
- any blocked dependency

## Output

- handoff-ready summary
- unresolved items
- next recommended starting point
