# Execute Plan

## Goal

Turn the feature plan into code and docs without losing the agreed scope.

## Steps

1. Follow the implementation steps in order.
2. Keep each change scoped to the current step.
3. Update canon/wiki when the plan changes materially.
4. Prepare the result for review and verification.

## Rules

- Do not silently expand scope.
- Escalate risky changes before taking them.
