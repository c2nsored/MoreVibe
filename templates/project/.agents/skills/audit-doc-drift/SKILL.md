# Audit Doc Drift

## Goal

Find documentation that no longer matches the project.

## Check

- canon vs code
- schema vs actual workflow
- handoff vs current status
- tasks vs actual completion state

## Output

- drift summary
- affected docs
- corrections needed
- urgency of update
