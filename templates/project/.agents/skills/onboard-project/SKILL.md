# Onboard Project

## Goal

Make the first session in a project productive instead of exploratory chaos.

## Steps

1. Read the project entrypoint and bootstrap docs.
2. Identify current goals and constraints.
3. Summarize the operating model.
4. Highlight missing context.
5. Recommend the first concrete action.

## Output

- project summary
- current goal
- important rules
- missing context
- first recommended action
