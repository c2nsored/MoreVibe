# Webapp State Review

## Goal

Spot hidden behavior issues in a web app before they turn into visible regressions.

## Focus

- state ownership
- loading and refresh behavior
- form or interaction drift
- UI state sync between components

## Output

- state risk summary
- affected components or screens
- missing verification
- recommended checks
