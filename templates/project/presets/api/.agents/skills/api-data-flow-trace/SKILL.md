# API Data Flow Trace

## Goal

Understand and verify how a change affects data movement across the backend.

## Focus

- route to service flow
- service to storage flow
- transformation points
- failure and edge-case handling

## Output

- traced flow summary
- affected layers
- risky transitions
- next verification step
