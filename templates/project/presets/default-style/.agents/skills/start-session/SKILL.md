# Start Session

## Goal

Recover enough context to work without guessing.

## Steps

1. Confirm the current workspace and target project.
2. Read `AGENTS.md` first.
3. Read `.morevibe/schema/SESSION_BOOTSTRAP.md`.
4. Review `.morevibe/wiki/state.md`, `.morevibe/canon/HANDOFF.md`, and `.morevibe/canon/TASKS.md`.
5. Restate the current goal and the scope you plan to touch.

## Rules

- Do not start implementation before the current state is restored.
- Prefer canon first, then wiki for acceleration.
