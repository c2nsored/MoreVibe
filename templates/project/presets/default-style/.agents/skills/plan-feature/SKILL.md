# Plan Feature

## Goal

Avoid starting non-trivial feature work without a clear plan.

## Steps

1. Restate the requested outcome.
2. Separate included scope from excluded scope.
3. Identify affected files, systems, docs, and risks.
4. Break the work into 2-5 implementation steps.
5. Decide whether subagents are needed.

## Output

- goal
- included scope
- excluded scope
- affected areas
- implementation steps
- risks
