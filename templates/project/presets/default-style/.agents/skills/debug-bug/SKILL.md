# Debug Bug

## Goal

Fix the real cause instead of guessing.

## Steps

1. Restate the observed issue.
2. Identify the expected behavior.
3. Gather reproduction steps and likely impact range.
4. Narrow the issue to the smallest responsible area.
5. Fix only after the likely root cause is clear.

## Output

- symptom
- expected behavior
- likely cause
- fix scope
