# Project Bootstrap

## Goal

Rebuild the operating context that the rest of the workflow depends on.

## Steps

1. Read `.morevibe/schema/PROJECT_SKILLS.md`.
2. Read `.morevibe/schema/SUBAGENT_ORCHESTRATION.md`.
3. Check `.morevibe/canon/PROJECT_OVERVIEW.md` for scope and boundaries.
4. Note the active workflow chain and any role split you expect to use.

## Output

- current workflow chain
- active role model
- current scope and constraints
