# Request Code Review

## Goal

Catch bugs, regressions, and risk before claiming completion.

## Focus

- broken behavior
- scope creep
- missing validation
- missing docs or handoff updates

## Rules

- Findings first, summary second.
- Prefer concrete file and behavior references.
