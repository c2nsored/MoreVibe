# Optional Commands

MoreVibe can expose command-style shortcuts in some tool integrations.

These commands are **optional accelerators**, not required workflow controls.

## Product Rule

- plain-language requests should be enough for normal use
- commands can help advanced users move faster
- users should not need to memorize command syntax to benefit from MoreVibe

## Example Positioning

Good positioning:

- "If you want a faster shortcut, you can use this command."
- "This command is optional."
- "Plain language should still work."

Bad positioning:

- "Use this command first or MoreVibe will not work correctly."
- "You must learn these commands before starting."

## Current Examples

- session start command
- session sync command

The right product experience is:

1. natural language first
2. optional shortcuts second
3. internal skill orchestration underneath
