# MoreVibe Finish Task

## Goal

Do not treat code edits alone as completion.

## Completion checklist

1. Change applied
2. Verification recorded
3. Review handled or explicitly skipped
4. Canon updated
5. Handoff updated
6. `morevibe-sync-memory` completed
7. `morevibe-lint-harness` run when the harness changed materially
8. Reusable outputs written back when appropriate
9. Final report prepared
