# MoreVibe Execute Plan

## Goal

Follow the plan instead of drifting into untracked changes.

## Steps

1. Restate the approved steps.
2. Mark which step is being executed now.
3. Limit changes to the current step's scope.
4. Record evidence that the step was completed.
5. Confirm the next step before moving on.

## Rules

- Do not silently add unplanned scope.
- If scope changes, update the plan first.
