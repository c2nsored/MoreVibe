# MoreVibe Update Docs

## Goal

Keep canon aligned with reality after changes.

## Steps

1. Identify what changed.
2. Find the authoritative canon files.
3. Update only the proper source of truth.
4. Avoid duplicating the same rule in multiple files.
5. Note which canon documents were updated.
