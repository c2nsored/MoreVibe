# MoreVibe Debug Bug

## Goal

Avoid guess-based fixes.

## Steps

1. Separate expected behavior from actual behavior.
2. Reproduce the issue when possible.
3. Inspect recent changes, logs, and relevant inputs.
4. Form a few root-cause hypotheses.
5. Verify the most likely cause.
6. Apply the smallest fix that addresses the confirmed cause.
7. Verify the fix and note remaining risks.
