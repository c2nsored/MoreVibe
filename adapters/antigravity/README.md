# Antigravity Adapter

This adapter now includes project integration assets based on Antigravity's global rule injection and CLI-driven execution model.

## Goal

Reuse the same MoreVibe core model while adapting installation and bootstrap behavior to Antigravity.

## Included documents

- `SPEC.md`
- `snippets/project-bootstrap.md`
- `snippets/global-bootstrap.md`
- `GLOBAL_CONFIG_TEMPLATE.md`
- `project/GEMINI.morevibe.md`
- `project/rules/*`

## Constraint

Antigravity support should not fork the MoreVibe knowledge model.

It should only adapt the integration layer.
