# MoreVibe Global Bootstrap for Antigravity

If a project contains `.morevibe/`, interpret it as a project-local MoreVibe harness.

Default rules:

- keep the project's root guide as the public entrypoint
- treat `.morevibe/canon/` as authoritative reference
- treat `.morevibe/wiki/` as compiled working memory
- use `morevibe-using-morevibe` as the main MoreVibe workflow router
- prefer backup and merge over overwrite when applying MoreVibe changes
